## 🔧 技術架構

### 後端 (Flask)
- `/set_state` - 設定狀態
- `/status` - 取得狀態
- `/agents` - 取得所有 AI 列表

### 前端 (Phaser.js + HTML)
- `index.html` - 主頁面
- `game.js` - 遊戲邏輯
- `layout.js` - 場景配置

### 資料格式
```json
{
  "agents": [
    {
      "agentId": "sakura",
      "name": "春野櫻",
      "state": "writing",
      "detail": "正在處理文件",
      "area": "writing",
      "updated_at": "2026-03-08T19:00:00Z"
    }
  ]
}
```
